# Walmart Weekly Sales Forecast (2012–2027)

## Project Summary
This project was created to demonstrate how historical retail sales data  (**I used data from 2010-2012**)can be used to forecast future demand and support business planning decisions. The focus was on building an end‑to‑end workflow that combines Python‑based forecasting with clear, stakeholder‑friendly visualization in Power BI.

## Project Outcome
- Built a Prophet‑based weekly sales forecasting model using historical Walmart data  
- Generated sales projections extending through 2027  
- Designed an interactive Power BI dashboard to visualize trends, seasonality, and forecasted demand
  
## Business Impact
The results of this project can support:
- **Inventory planning** by anticipating high‑demand periods  
- **Staffing decisions** by aligning labor with forecasted sales  
- **Strategic planning** through better visibility into long‑term demand trends  

## This approach reflects how forecasting is commonly used in retail and supply‑chain analytics.

-This project analyzes historical Walmart weekly sales data (2010–2012) to forecast demand through 2027.
-The results are visualized in an interactive Power BI dashboard to support inventory and staffing planning decisions.

**Tools Used:** Python (Pandas, Prophet), Power BI

## Main Deliverables
- ✅ Interactive Power BI Dashboard (primary output)
- ✅ Python notebooks for data cleaning, analysis, and forecasting
- ✅ Cleaned dataset used for modeling (no raw data included)
